## Patches
- Vertcenter
- Scrollback
- Font2
- Scrollback with key bindings/mouse wheel/vim bindings
- Zoom/change font size
- Copy/Paste text
- New terminal in cwd

## Unique features (using dmenu)
- Follow urls: `alt-l`
- Copy urls: `alt-y`
- Copy command output: `alt-o`
